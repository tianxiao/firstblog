Maps and Homeomorphisms

Two metric spaces are considered to be the same if the points that comprise them are connected the same way.
Homeomorphisms is a important concept to the topology theory we can use the easy representation to sutdy the topology space.
Continuity is just step on the way to topological equivalence, because a continuous function can map many points to a single point in the target space like the y=x^2+y^2 map and true equivalence is marked by a homeomorphisms a one-to-one function from one space to another that possesses both continuity and a continuous inverse, so that the limit points are preserved in both directions.
